Task 09: Syracuse Crime Geospatial Analysis

This phase focuses on mapping crime incidents geographically using latitude and longitude fields from the Part 1 Offenses dataset. Using the Folium library, a heatmap and cluster map were created to show high-density zones.

## Contents
geospatial_mapping.py: Script to create interactive maps using Folium
map_summary.md: Interpretation of spatial crime trends and clusters
.gitignore: Best practices for excluding data/API keys from GitHub
